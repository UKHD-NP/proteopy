``.ann``
========

.. module:: proteopy.ann
   :synopsis: Annotation functions for samples and variables

The ``proteopy.ann`` module provides functions to annotate samples (``.obs``) and
variables (``.var``) in :class:`~anndata.AnnData` objects with metadata from
external sources.
