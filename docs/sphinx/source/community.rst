Community
=========

ProteoPy is developed and maintained by the BludauLab in the Department of
Computational `Neuropathology
<https://www.klinikum.uni-heidelberg.de/pathologisches-institut/neuropathologie>`_,
`University Hospital Heidelberg <https://www.klinikum.uni-heidelberg.de/>`_.

Questions & Discussions
-----------------------

For general questions, usage discussions, and community support, please use
`GitHub Discussions <https://github.com/UKHD-NP/proteopy/discussions>`_.

Bug Reports & Feature Requests
------------------------------

If you encounter a bug or have a feature request, please open an issue on
`GitHub Issues <https://github.com/UKHD-NP/proteopy/issues>`_.

When reporting bugs, please include:

- ProteoPy version (``proteopy.__version__``)
- Python version
- Operating system
- Minimal reproducible example
- Full error traceback

Related Projects
----------------

ProteoPy integrates with the broader scientific Python ecosystem:

- `AnnData <https://anndata.readthedocs.io/>`_ - Annotated data structures
- `scanpy <https://scanpy.readthedocs.io/>`_ - Single-cell analysis
- `MuData <https://mudata.readthedocs.io/>`_ - Multi-modal data structures
- `MUON <https://muon.readthedocs.io/>`_ - Multi-omics analysis
- `squidpy <https://squidpy.readthedocs.io/>`_ - Spatial omics analysis
